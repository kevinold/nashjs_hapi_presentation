
The slide are via [reveal.js](https://github.com/hakimel/reveal.js).

### To view slides
1.  
   ```sh
   $ npm install
   ```
2.  
   ```sh
   $ grunt serve
   ```
3.  
   ```sh
   $ open http://localhost:8000
   ```
